# Provided Code 

A provided code is a **File** that contains a code.

## Example 1

**File Name**: main.c

```
#include <stdio.h>
#include <add.h>
int main() {
    int a, b;
    scanf("%d %d", a, b);
    printf(add(a, b));
    return 0;
}
```
